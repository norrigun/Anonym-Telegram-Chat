# Anonym-Telegram-Chat
# :warning:Установка:
1. Скачать файлы бота
2. Закинуть на сервер/ПК без кириллицы в пути
3. Создать Telegram бота в @BotFather
4. Отредактировать файл *.env*
Telegram ID администраторов прописываются через запятую без пробелов

# :bulb:Запуск:
1. <pre>pip install -r requirements.txt</pre>
2. <pre>pip install "python-telegram-bot[job-queue]"</pre>
3. <pre>py bot.py</pre>
